## Description
Our `string.h` C library implementation with Unit tests.

## Build
      $ git clone https://github.com/ksilisk/string.h.git
      $ cd string.h
      $ make
      
## Tests
Unit tests with `check.h` C library.

      $ cd string.h
      $ make test
      $ ./a.out
